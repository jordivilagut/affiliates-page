# Digital Bio

A personal digital bio built with React, TypeScript, and Vite.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
# or
yarn
```

### Development

Start the development server:
```bash
npm run dev
# or
yarn dev
```

### Building for Production

Build the project:
```bash
npm run build
# or
yarn build
```

Preview the production build:
```bash
npm run preview
# or
yarn preview
```

## Tech Stack

- React
- TypeScript
- Vite
- ESLint
